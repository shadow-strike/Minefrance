Minefrance
===========

Serveur Minecraft sous craftbukkit 1.2.5

Site: [http://minefrance.com](http://minefrance.com)  
Bugs/Suggestions: [http://minefrance.com/2.0/forum](http://minefrance.com/2.0/forum)

Compilation
-----------

Nous utilisons maven pour gérer nos dépendances.

* Installer [Maven 3](http://maven.apache.org/download.html)
* Télécharger et installez le [Serveur](https://github.com/shadow-strike/Minefrance)
    * * Note*: ce n'est pas nécessaire d'utiliser nos sources,vous pourriez avoir une plus récente (avec vos propres modifications: D)
* Télécharger et lancer en: `mvn clean package`
